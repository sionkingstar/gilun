# 2026 길운 달력 - 사주명가 대운

개인 맞춤형 2026년 길운 달력 서비스입니다.

## 📋 프로젝트 개요

- **목적**: 2026년 단일 연도 개인 길운 달력 제공
- **핵심 기능**: 일별 일진(간지) 표시, 길운 판단, 특정 이슈(사업운/이동수 등) 안내
- **대상**: 사주 상담 고객

## ✅ 완성된 기능

### 1. 고객용 달력 페이지 (`index.html`)
- 생년월일 8자리 토큰으로 접근 (예: 19771006)
- **사주팔자 4기둥 시각화** (년주/월주/일주/시주)
- **현재 대운 정보** 및 진행률 표시
- **2026년 세운** 요약 정보
- **12개월 월별 달력 그리드** (일~토)
  - 각 날짜에 **일진(간지)** 표시
  - **길운 표시** (★대길 / ◎길 / ○평 / △주의)
  - **일별 특정 이슈 아이콘** 표시
    - 💰 재물운
    - 📈 사업운
    - ✈️ 이동수
    - 🤝 귀인운
    - 📝 문서운
    - ⭐ 인기운
    - 📚 학습운
    - 🤲 협력운
    - ⚠️ 건강주의
- **월별 요약 블록** (키워드, 조언, 이슈별 날짜 정리)
- **PDF 다운로드** 기능

### 2. 관리자 페이지 (`admin.html`)
- **비밀번호 로그인** 보안 (기본: `admin2026`)
- 고객 등록/수정/삭제
- **토큰 자동 생성** (생년월일 YYYYMMDD 형식)
- 고객별 전용 URL 링크 복사
- **PDF 직접 생성** 기능
- 고객 목록 검색

### 3. 사주 계산 로직 (`js/saju-calendar.js`)
- **60갑자 일진 자동 계산** (2026년 전체)
- **길운 판단 규칙**:
  - 월운 신살 영향 (합/충/도화살/화개살 등)
  - 월운 십성 영향 (재성/관성/인성/식상)
  - 일간과 일진 천간의 오행 관계
  - 특수 길일 판단 (갑자, 병오, 무진 등)
  - 천간충 주의일 판단
- **일별 특정 이슈 판단**:
  - 재물운: 일간이 극하는 오행일
  - 사업운: 재성월 + 토(土) 일진
  - 이동수: 역마지지(寅申巳亥) 일
  - 귀인운: 인성 일진
  - 문서운: 정인/편인월 + 특정 지지
  - 인기운: 도화살월 + 도화지지(子午卯酉)
  - 학습운: 화개살월 + 화개지지(辰戌丑未)

## 📊 데이터 모델

### customers 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| name | text | 고객 이름 |
| birth_info | text | 생년월일시 (예: 1977-10-06 08:30) |
| gender | text | 성별 |
| token | text | 접근 토큰 (YYYYMMDD) |
| saju_data | text | 사주 JSON 데이터 |
| memo | text | 메모 |

### saju_data JSON 구조
```json
{
  "daeun": { "direction": "순행", "data": [...] },
  "pillar": { "data": [...] },
  "sewun": { "data": [{ "연도": 2026, "월운": [...] }] },
  "user_info": { "일간(본인)": "丙(火)", ... }
}
```
